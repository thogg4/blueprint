BluePrint
=========

Uses
----

* Postgresql
* JQuery
* SCSS
* Bootstrap
* Minitest
* Factory Girl
* Unicorn

Any of these can be changed pretty easily.

Usage
-----

* Clone to folder with name you choose for project
```
git clone git@github.com:nwalkingshaw/blueprint.git [FOLDER_NAME]
```

* Change config/database.yml file
```
rake db:create:all  
```
* Customize!!
